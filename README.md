# Institute of Digital Risk (IDR) - Technical Assessment

## Project Overview
This project is a responsive, single-page website for the "Institute of Digital Risk" (IDR), an industry-led training and deployment institute. It was built using semantic HTML5, CSS3 (utilizing Flexbox and CSS Grid), and vanilla JavaScript to ensure optimal performance without reliance on external frameworks. 

## Design & Branding Note
The IDR logo utilizes a cube-inspired geometric shape to symbolize structure, digital resilience, and the multi-faceted nature of technological risk. The primary color palette of vivid orange against deep black and white establishes a clean, minimalist aesthetic that balances an authoritative tech feel with modern educational accessibility. Typography pairs the bold, structured 'Syne' font for headings to convey strength, while 'DM Sans' ensures high legibility and a welcoming tone for the body text.

## Technical Specifications
* **Structure:** Separated into `index.html`, `style.css`, and `script.js` for clear separation of concerns.
* **Responsiveness:** Fluid scaling utilizing CSS clamp(), Flexbox, and Grid, ensuring perfect layout integrity across desktop, tablet, and mobile breakpoints.
* **Interactivity:** Implemented a sticky navigation bar with a glassmorphism effect, smooth scrolling to sections, and a functional mobile hamburger menu.
* **Accessibility:** Semantic HTML tags (`<header>`, `<nav>`, `<section>`, `<article>`) and appropriate ARIA labels used throughout.

## How to Run
1. Unzip the project folder.
2. Double-click the `index.html` file to open it in your default web browser. No local server or build tools are required.
